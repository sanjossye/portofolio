About me..

Saya adalah mahasiswa di Universitas Sumatera Utara dengan program studi Teknologi Informasi. Sebelumnya, saya adalah lulusan dari SMK dengan jurusan Teknologi Jaringan Komputer dan Telekomunikasi (TJKT), memberikan saya dasar yang kuat dalam teknologi dan jaringan.

Dengan pengalaman yang mendalam di bidang jaringan, saya telah terlibat dalam berbagai proyek yang memperkuat keterampilan teknis saya, termasuk instalasi dan pemeliharaan infrastruktur jaringan.

Experience

Konfigurasi Virtual

Saya memiliki pengalaman dalam remote configuration Optical Network Unit (ONU), yang memungkinkan saya untuk mengelola setting jaringan seperti VLAN, DHCP, dan IP address tanpa perlu intervensi fisik. Kemampuan ini memungkinkan saya untuk memberikan solusi yang cepat dan efektif.

Server Upkeep

Server Upkeep dalam konteks jaringan adalah proses strategis yang memastikan kinerja server tetap optimal dan aman. Ini mencakup pemantauan real-time terhadap resource seperti CPU dan bandwidth, penerapan security patches untuk melindungi dari ancaman.

Perawatan Infrastruktur

Perawatan Infrastruktur Jaringan adalah langkah krusial untuk memastikan keandalan dan performa optimal dari sistem jaringan. Saya fokus pada pemeliharaan komponen penting seperti Optical Distribution Point (ODP) dan Optical Drop Cable (ODC). dan perangkat infrakstruktur lainnya.

Narasumber

Sebagai narasumber di workshop MC, saya berbagi pengetahuan tentang teknik penguasaan panggung dan komunikasi efektif. Dalam acara tersebut, saya membahas pentingnya keterampilan interpersonal dan pengelolaan waktu, serta melibatkan peserta dalam praktik langsung.

FTTH (Fiber to the Home)

Dalam perjalanan saya mendalami dunia FTTH (Fiber to the Home), saya telah memperoleh pemahaman yang mendalam tentang teknologi jaringan fiber optik dan implementasinya. Saya terlibat dalam berbagai proyek yang melibatkan perencanaan dan pemasangan.

Instalasi Mikrotik

Saya mahir dalam instalasi Mikrotik, termasuk konfigurasi router dan troubleshooting untuk memperbaiki masalah koneksi. Selain itu, saya memiliki keahlian dalam blocking konten dan pengelolaan bandwidth untuk meningkatkan efisiensi jaringan.
